    Class DmvicClient

    Main DMVIC API client.

    Handles authentication, token caching, request signing, and all DMVIC V4 API operations. Supports X.509 mutual-TLS when certificates are provided.

    Example

    const client = new DmvicClient({
      environment: 'production',
      credentials: { username: 'user', password: 'pass' },
      clientId: 'my-client-id',
    });
    await client.login();
    const cert = await client.getCertificate('CERT-001');
